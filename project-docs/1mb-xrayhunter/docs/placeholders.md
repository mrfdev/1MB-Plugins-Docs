# 1MB-XRayHunter Placeholders

1MB-XRayHunter does not currently register PlaceholderAPI placeholders.

That means:

- there are no `%xrayhunter_*%` placeholders to configure yet
- there is no PlaceholderAPI expansion jar for this project
- there are no documented scoreboard, hologram, or chat placeholders for this plugin right now

If placeholder support is added in a future release, this page should be expanded with:

- exact placeholder names
- sample output
- value meanings
- refresh/update behavior
