# Placeholders

HeadDisplays has a soft dependency on PlaceholderAPI.

## PlaceholderAPI In Display Text

When PlaceholderAPI is present, HeadDisplays calls PlaceholderAPI to resolve placeholders in display text. This means staff can use PlaceholderAPI placeholders supported by other installed plugins inside HeadDisplays text.

## Plugin-Owned Placeholders

No HeadDisplays-owned PlaceholderAPI expansion placeholders were verified from this jar. The inspected `PlaceholderManager` only delegates text through PlaceholderAPI when PlaceholderAPI is enabled.

## Operational Notes

`/hd info` reports whether PlaceholderAPI was found by the plugin.
