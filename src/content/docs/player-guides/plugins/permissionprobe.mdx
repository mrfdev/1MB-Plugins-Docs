This page introduces PermissionProbe from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Owner-only read-only permission diagnosis, passive denial recording, command checks, feature access summaries, server group source traces, and Bukkit metadata.

## How Players Use It

Use this page as a friendly guide to PermissionProbe. It explains what the feature is for, what players can try in-game, and which commands are available when your account has access.
