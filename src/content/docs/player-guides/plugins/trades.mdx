This page introduces Trades from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

/trade GUI exchanges for item, money, and EXP hand-ins, migrated from the old standalone 1MBTrades plugin.

## How Players Use It

Start in-game with /trade. The sections below explain what Trades is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Open or view Trades with /trade.

## Quick Start

1. Start with `/trade` to open or view Trades.
2. Use `/trade info` when you want the in-game help or a short explanation.
3. Try `/trade help` when you are ready to use one of the feature actions.

## Examples

- `/trade`
- `/trade info`
- `/trade open summer_event`
- `/trade open category vote`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
