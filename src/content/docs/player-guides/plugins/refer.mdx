## How Players Use It

Start with /refer. The sections below explain what Refer is for, which commands players can try, and what to expect when your account has access.

Available features include:

- A two-step referral request and confirmation flow between two players.
- Playtime eligibility checks for both the referrer and the referred player.
- Duplicate-claim protection and expiring pending requests.
- Separate configured rewards or permission markers for each participant.
- Status, accept, deny, and cancel controls for an active referral request.

## Quick Start

1. Start with `/refer` to open or view Refer.
2. Use `/refer status` to check your current progress, settings, or state.
3. Use `/refer info` when you want the in-game help or a short explanation.
4. Try `/refer Steve` when you are ready to use one of the feature actions.
