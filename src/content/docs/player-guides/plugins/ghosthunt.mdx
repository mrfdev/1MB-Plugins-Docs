This page introduces Halloween Ghost Hunt from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

/ghosthunt themed edition in the shared CoconutHunt jar with event-bound Ghost heads, private positional effects, accessibility modes, Haunting Chains, historical claims, guarded activation, immutable snapshots, and isolated debug data.

## How Players Use It

Run /ghosthunt, then use Visit the Halloween world to travel to the event. Right-click an unlocked Ghost head with the main hand. Left-clicking is not a discovery action. Every Ghost stays in the world for other players and counts only once per player for that edition.

## Quick Start

1. Start with `/ghosthunt` to open or view Halloween Ghost Hunt.
2. Use `/ghosthunt info` when you want the in-game help or a short explanation.
3. Try `/ghosthunt help` when you are ready to use one of the feature actions.
