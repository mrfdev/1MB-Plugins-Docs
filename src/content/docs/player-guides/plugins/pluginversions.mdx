This page introduces PluginVersions from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Plugin version inventory, tested Java/server/dependency gate, URL curation, and Markdown/Discord export reports migrated from the standalone PluginVersions project.

## How Players Use It

Start in-game with /pv. The sections below explain what PluginVersions is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Open or view PluginVersions with /pv.
- Browse available PluginVersions entries with /pv list.

## Quick Start

1. Start with `/pv` to open or view PluginVersions.
2. Use `/pv info` when you want the in-game help or a short explanation.
3. Try `/pv help` when you are ready to use one of the feature actions.

## Examples

- `/pv info`
- `/pv list`
- `/pv list 2`
- `/pv gate`
- `/pv set database.update-on-list false`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- Top lists and leaderboards are public-facing progress views when the feature enables them.
- Settings commands only change your own player preferences unless the page says otherwise.
