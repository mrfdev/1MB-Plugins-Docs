This page introduces 1MBStaffMsg from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Runtime recent history helper for server staffmsg and /s recent.

## How Players Use It

Start in-game with /s recent. The sections below explain what 1MBStaffMsg is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Adds /s recent [page], /staffmsg recent [page], /cmi staffmsg recent [page], and /1mbstaffmsg recent [page].
- Adds recent as a tab suggestion for /s, /staffmsg, and /cmi staffmsg when the sender can use server staffmsg.
- Recent rows are compact.
- Clicking a recent sender suggests /s for a quick staffmsg reply.

## Quick Start

1. Start with `/s recent` to open or view 1MBStaffMsg.
2. Use `/1mbstaffmsg status` to check your current progress, settings, or state.
3. Use `/1mbstaffmsg info` when you want the in-game help or a short explanation.
4. Try `/staffmsg recent` when you are ready to use one of the feature actions.
