This page introduces WorthDrift from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Server sell-event drift analyzer for worth values and economy-loop signals.

## How Players Use It

Start in-game with /worthdrift. The sections below explain what WorthDrift is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Read per-material sold amounts and payments from server's sell event.
- Flag player concentration when one player accounts for a large share of a material's observed payment.
- Ask server WorthItem whether an exploit recipe is known for a material.
- Keep a bounded recent sample list for report context.
- Write timestamped support reports to the shared cache folder.

## Quick Start

1. Start with `/worthdrift` to open or view WorthDrift.
2. Use `/worthdrift status` to check your current progress, settings, or state.
3. Try `/worthdrift top` when you are ready to use one of the feature actions.

- `/worthdrift top`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- You may also see /wdrift as a shortcut.
- Top lists and leaderboards are public-facing progress views when the feature enables them.
