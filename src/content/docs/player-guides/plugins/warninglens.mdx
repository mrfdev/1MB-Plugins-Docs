This page introduces WarningLens as a public staff guide. It focuses on what the tool does, how authorized staff get started, and which commands are useful.

## What It Does

Read-only warning, jail, ban, kick, and unban visibility.

## How Staff Use It

Start with /warninglens status. The sections below explain what WarningLens is for, which commands authorized staff can use, and what its output means.

Available features include:

- Inspect a CMI player's warning count and warning points.
- Split inspection into summary, punishment, warning, or full views.
- Show current CMI ban, jail, mute, shadow mute, cuff, and alert state.
- List the player's CMI warnings with category, points, actor, time, and reason when permitted.
- Observe recent CMI warn, jail, ban, kick, and unban events during this server runtime.
- Filter recent moderation events by type.
- Show total and recent moderation event counts by type.
- Search recent events with strict input and reason-aware privacy rules.

## Quick Start

1. Start with `/warninglens status` to open or view WarningLens.
2. Use `/warninglens status` to check your current progress, settings, or state.
3. Try `/warninglens Steve` when you are ready to use one of the feature actions.
