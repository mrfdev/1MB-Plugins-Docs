This page introduces WarningLens from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Read-only warning, jail, ban, kick, and unban visibility.

## How Players Use It

Start in-game with /warninglens status. The sections below explain what WarningLens is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Inspect a server player's warning count and warning points.
- Show current server ban, jail, mute, shadow mute, cuff, and alert state.
- List the player's server warnings with category, points, actor, time, and reason when permitted.
- Observe recent server warn, jail, ban, kick, and unban events during this server runtime.
- Filter recent moderation events by type.
- Show total and recent moderation event counts by type.
- Search recent events with strict input and reason-aware privacy rules.
- Dump recent events to the WarningLens cache folder for support review.

## Quick Start

1. Start with `/warninglens status` to open or view WarningLens.
2. Use `/warninglens status` to check your current progress, settings, or state.
3. Try `/warninglens Steve` when you are ready to use one of the feature actions.
