This page introduces ScheduledTips from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Clickable rotating tips with per-player dismiss, visibility controls, and live booster reminders.

## How Players Use It

Start in-game with /tips status. The sections below explain what ScheduledTips is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Send clickable chat messages with subtle MiniMessage styling.
- Add a red [x] click action before a tip after a player has seen it enough times.
- Add a [tips off] click action so players can globally hide scheduled tips.
- Make /tips on and /tips off confirmation messages clickable so players can quickly switch back.
- Replace matching mcMMO, Jobs, /points, and DiscordChat tips with a live booster reminder when the matching /rate booster is active.
- Track per-player seen counts, dismissed tips, and global visibility in shared playerdata.

## Quick Start

1. Start with `/tips status` to open or view ScheduledTips.
2. Use `/tips status` to check your current progress, settings, or state.
3. Try `/tips on` when you are ready to use one of the feature actions.

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- Settings commands only change your own player preferences unless the page says otherwise.
