This page introduces DiscordChat from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

DiscordChat is a player-fun feature plugin for turning the DiscordSRV #server-chat bridge into a gentle engagement loop. Linked players earn DiscordChat EXP from meaningful chat, emotes, and daily participation. Every complete 25 EXP converts into one spendable point by default, while streak...

## How Players Use It

Start with /discordchat. The sections below explain what DiscordChat is for, which commands players can try, and what to expect when your account has access.

Available features include:

- DiscordChat EXP for meaningful linked-account activity in the DiscordSRV server-chat bridge.
- Automatic floor conversion of complete EXP units into spendable points, with streak milestones adding separate bonus points.
- A GUI with personal status, tracked activity, community pulse, milestones, history, and public top lists.
- Two-step point reward confirmations, per-reward cooldowns, and safe point-funded item tools.
- Temporary community XP bonus windows that can also appear in /rate.
- Unlockable chat prefix and suffix presets.
- Personal controls for Discord reminders and public milestone celebrations.
- Linked-account guidance, first-Discord participation feedback, and privacy-aware progress tracking.

## Quick Start

1. Start with `/discordchat` to open or view DiscordChat.
2. Use `/discordchat status` to check your current progress, settings, or state.
3. Try `/discordchat top` when you are ready to use one of the feature actions.

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- Top lists and leaderboards are public-facing progress views when the feature enables them.
- Settings commands only change your own player preferences unless the page says otherwise.
