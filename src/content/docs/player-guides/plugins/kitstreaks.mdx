This page introduces KitStreaks from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

KitStreaks helps players follow kit claim streaks and milestone rewards. It is meant for players who like daily or recurring kit progress and want to see what they have already claimed.

## How Players Use It

Start in-game with /kitstreak info. The sections below explain what KitStreaks is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Tracks successful server kit claims through the serverUserKitAcquireEvent.
- Supports a whitelist of tracked kits, exact ignored kits, and ignored kit prefixes for plugin-delivered reward kits.
- Default tracks are prepared for daily, starter, and events.
- Default milestones include 7, 14, 21, 28, and 365 day style goals.
- Shows streak health as safe today, due today, grace, or missed.
- Supports optional milestone reward commands.
- Supports claim-based rewards or automatic reward execution on earn.
- The feature is enabled

## Quick Start

1. Start with `/kitstreak info` to open or view KitStreaks.
2. Use `/kitstreak status` to check your current progress, settings, or state.
3. Use `/kitstreak info` when you want the in-game help or a short explanation.
4. Try `/kitstreak tracks` when you are ready to use one of the feature actions.

- `/kitstreak info`
- `/kitstreak status`
- `/kitstreak status daily`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- You may also see /kitstreaks or /kstreak as a shortcut.
- Top lists and leaderboards are public-facing progress views when the feature enables them.
