This page introduces FilterLab from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Safe server ChatFilter.yml rule simulation before live use.

## How Players Use It

Start in-game with /filterlab. The sections below explain what FilterLab is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Shows server ChatFilter enabled state, deny rule count, enabled rule count, and caps filter settings.
- Simulates staff-provided text against loaded deny rules.
- Tracks server chat, server chat filter, and server caps filter event counts.
- Lists loaded deny rules with enabled state, group, block type, and regex count.
- Keeps bounded recent local simulations for the current runtime.

## Quick Start

1. Start with `/filterlab` to open or view FilterLab.
2. Use `/filterlab status` to check your current progress, settings, or state.
3. Try `/filterlab rules` when you are ready to use one of the feature actions.
