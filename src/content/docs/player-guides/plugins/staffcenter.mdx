This page introduces StaffCenter from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

The first version should avoid punishment actions and focus on safely displaying useful server-backed player context in one place.

## How Players Use It

Start in-game with /staffcenter Steve. The sections below explain what StaffCenter is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Check your own StaffCenter progress or settings with /staffcenter status.

## Quick Start

1. Start with `/staffcenter Steve` to open or view StaffCenter.
2. Use `/staffcenter status` to check your current progress, settings, or state.
3. Try `/staffcenter Steve` when you are ready to use one of the feature actions.

## Examples

- `/staffcenter Notch`
- `/staffcenter recent`
- `/staffcenter stats`
- `/staffcenter find Notch`
- `/staffcenter dump all`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
