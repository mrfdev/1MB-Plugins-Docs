This page introduces Exchange from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

/exchange GUI exchanges for item, money, and EXP hand-ins, migrated from the old standalone 1MBTrades plugin.

## How Players Use It

Start with /exchange. The sections below explain what Exchange is for, which commands players can try, and what to expect when your account has access.

Available features include:

- A menu of configured exchanges for seasonal events, vote-token hand-ins, kit unlocks, and resource turn-ins.
- Trades that can require exact items, money, EXP levels, or a combination of requirements.
- Command-based rewards that run only after the exchange requirements pass.
- One-time, repeatable, and limited-use exchanges with per-player usage tracking.
- Category pages, date windows, world restrictions, and permission-gated entries when configured.
- Optional hiding of completed exchanges so the normal menu stays focused on usable trades.

## Quick Start

1. Start with `/exchange` to open or view Exchange.
2. Use `/exchange info` when you want the in-game help or a short explanation.
3. Try `/exchange help` when you are ready to use one of the feature actions.

## Examples

- `/exchange`
- `/exchange info`
- `/exchange open summer_event`
- `/exchange open category vote`
- `/exchange set display summer_event <#F6D365><bold>Summer Event Trade</bold></#F6D365>`
- `/exchange set description summer_event Bring your event tokens here.`
- `/exchange set permission summer_event onembcmi.exchange.summer_event`
- `/exchange set completion summer_event onembcmi.exchange.completed.summer_event`
- `/exchange set max summer_event 1`
- `/exchange set worlds summer_event survival,oneblock`
- `/exchange set money summer_event 2500`
- `/exchange set exp summer_event 5`
- `/exchange set start summer_event 06-01-2026`
- `/exchange set end summer_event 08-31-2026`
- `/exchange toggle summer_event false`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- Settings commands only change your own player preferences unless the page says otherwise.
