This page introduces UpdateSmoke from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Post-update smoke checks for hooks, commands, placeholders, feature metadata, and key server library APIs.

## How Players Use It

Start in-game with /updatesmoke. The sections below explain what UpdateSmoke is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Open or view UpdateSmoke with /updatesmoke.
- Check your own UpdateSmoke progress or settings with /updatesmoke status.

## Quick Start

1. Start with `/updatesmoke` to open or view UpdateSmoke.
2. Use `/updatesmoke status` to check your current progress, settings, or state.
3. Use `/updatesmoke help` when you want the in-game help or a short explanation.
4. Try `/updatesmoke run` when you are ready to use one of the feature actions.
