This page introduces CMIConfig from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Owner GUI for selected server library config toggles with backups and reload prompts.

## How Players Use It

Start in-game with /cmiconfig. The sections below explain what CMIConfig is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Per-file pages show discovered boolean toggles with on/off dye indicators.
- Click a toggle to open a confirmation screen.
- Creates a timestamped backup next to the edited YAML file before writing.
- Uses a targeted line edit so server's comments and most formatting are preserved.

## Quick Start

1. Start with `/cmiconfig` to open or view CMIConfig.
2. Use `/cmiconfig status` to check your current progress, settings, or state.
3. Try `/cmiconfig open` when you are ready to use one of the feature actions.
