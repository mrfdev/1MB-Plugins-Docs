This page introduces CMIPlaceholderCheck from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Server-style placeholder lookup catalog with live example parsing.

## How Players Use It

Start in-game with /cmiplaceholders. The sections below explain what CMIPlaceholderCheck is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Open or view CMIPlaceholderCheck with /cmiplaceholders.
- Check your own CMIPlaceholderCheck progress or settings with /cmiplaceholders status.
- Browse available CMIPlaceholderCheck entries with /cmiplaceholders list.

## Quick Start

1. Start with `/cmiplaceholders` to open or view CMIPlaceholderCheck.
2. Use `/cmiplaceholders status` to check your current progress, settings, or state.
3. Use `/cmiplaceholders help` when you want the in-game help or a short explanation.
4. Try `/cmiplaceholders check cheer` when you are ready to use one of the feature actions.

## Aliases

- `/cmipholders`

## Examples

- `/cmiplaceholders status`
- `/cmiplaceholders check cmi`
- `/cmiplaceholders check balance`
- `/cmiplaceholders check onembcmi`
- `/cmiplaceholders list`
- `/cmiplaceholders list 2`
- `/cmiplaceholders plugin CMI`
- `/cmiplaceholders plugin PlaceholderAPI`
- `/cmiplaceholders plugin onembcmi`
- `/cmiplaceholders example %cmi_user_name% mrfloris`
- `/cmiplaceholders example %onembcmi_global.status.loaded%`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- You may also see /cmipholders or /checkplaceholders as a shortcut.
