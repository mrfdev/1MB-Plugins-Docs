This page introduces GameTypes from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

GameTypes helps players find the correct menu for island-style game modes. Use it when you are in a BentoBox world or when you want the server to show the available game type menus.

## How Players Use It

Start with /gametype. The sections below explain what GameTypes is for, which commands players can try, and what to expect when your account has access.

Available features include:

- One menu system for OneBlock, SkyBlock, AcidIsland, CaveBlock, and SkyGrid.
- Automatic current-world detection with a game-type index when no island world is detected.
- Core island actions for travel, creation, teams, information, help, top lists, kits, and daily shops.
- Addon buttons for warps, challenges, levels, biomes, generators, borders, and other installed BentoBox features.
- Per-game-type button visibility based on installed and enabled BentoBox addons.
- A confirmation screen for destructive island reset actions when reset is enabled.
- Back-to-index navigation for switching between configured game types.

## Quick Start

1. Start with `/gametype` to open or view GameTypes.
2. Use `/gametype status` to check your current progress, settings, or state.
3. Use `/gametype info` when you want the in-game help or a short explanation.
4. Try `/gametype menu` when you are ready to use one of the feature actions.

## Examples

- `/gametype`
- `/gametype menu`
- `/gametype menu oneblock`
- `/gametype status`
- `/gametype info`
- `/gametype help`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
