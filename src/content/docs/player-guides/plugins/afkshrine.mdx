This page introduces AFKShrine from a player point of view. It focuses on what you can do in-game, how to get started, and which commands are useful when the feature is available to you.

## What It Does

AFKShrine is a player-fun plugin that turns server AFK state into a small visible experience. When a player becomes AFK, the plugin can show a subtle shrine effect around them with soft dust particles, optional sparkles, a private boss bar, and optional text that explains they are away.

## How Players Use It

Start in-game with /afkshrine. The sections below explain what AFKShrine is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Open or view AFKShrine with /afkshrine.
- Check your own AFKShrine progress or settings with /afkshrine status.
- View public AFKShrine rankings with /afkshrine top.

## Quick Start

1. Start with `/afkshrine` to open or view AFKShrine.
2. Use `/afkshrine status` to check your current progress, settings, or state.
3. Try `/afkshrine toggle` when you are ready to use one of the feature actions.

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- Top lists and leaderboards are public-facing progress views when the feature enables them.
- Settings commands only change your own player preferences unless the page says otherwise.
