This page introduces PvPToggle from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

PvPToggle gives players a clear /pvp switch. Use it to see whether PvP is enabled for you and to turn your own PvP state on or off when the server allows it.

## How Players Use It

Start with /pvp. The sections below explain what PvPToggle is for, which commands players can try, and what to expect when your account has access.

Available features include:

- A personal PvP on/off switch when the current world and server rules allow it.
- A clear status view for the current PvP setting and protection state.
- Direct /pvp on, /pvp off, and /pvp toggle choices.
- Configurable feedback and visual effects when the state changes.

## Quick Start

1. Start with `/pvp` to open or view PvPToggle.
2. Use `/pvp status` to check your current progress, settings, or state.
3. Use `/pvp help` when you want the in-game help or a short explanation.
4. Try `/pvp info` when you are ready to use one of the feature actions.

## Good To Know

- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
