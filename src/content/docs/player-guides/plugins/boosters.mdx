This page introduces Boosters from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Boosters lets players check the current server booster status from one simple place. Use it when you want to know whether skill, job, points, or event multipliers are active before you start grinding.

## How Players Use It

Start in-game with /rate. The sections below explain what Boosters is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Open /rate to see which boosters are currently active.
- Check /rate status for a quick text summary.
- Use /rate info or /rate help when you want a short explanation in-game.

## Quick Start

1. Start with `/rate` to open or view Boosters.
2. Use `/rate status` to check your current progress, settings, or state.
3. Use `/rate info` when you want the in-game help or a short explanation.
4. Try `/rate help` when you are ready to use one of the feature actions.
