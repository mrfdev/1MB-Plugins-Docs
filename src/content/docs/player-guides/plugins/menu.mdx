This page introduces Menu from a player point of view. It focuses on what you can do in-game, how to get started, and which commands are useful when the feature is available to you.

## What It Does

Configurable /menu GUI for server discovery, travel, economy, progression, and server ctext info.

## How Players Use It

Players run /menu to open the first page. Each button shows its title, a short description, the direct command in gray, and a gold left-click hint. That lets players discover features visually while also learning the normal command they can type later.

## Things You Can Do

- Open or view Menu with /menu.
- Check your own Menu progress or settings with /menu status.

## Quick Start

1. Start with `/menu` to open or view Menu.
2. Use `/menu status` to check your current progress, settings, or state.
3. Use `/menu info` when you want the in-game help or a short explanation.
4. Try `/menu 1` when you are ready to use one of the feature actions.
