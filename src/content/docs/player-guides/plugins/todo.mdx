Players can add quick notes while playing:

## Commands

<table>
  <thead>
    <tr>
      <th>Command</th>

    <tr>
      <td><code>/todo</code></td>
      <td>Opens or shows the main PlayerTodo player view.</td>
      <td><code>/todo</code></td>
    </tr>
    <tr>
      <td><code>/todo &lt;text&gt;</code></td>
      <td>Uses PlayerTodo with the value you provide.</td>
      <td><code>/todo Build a starter house</code></td>
    </tr>
    <tr>
      <td><code>/todo add &lt;text&gt;</code></td>
      <td>Runs the PlayerTodo add action.</td>
      <td><code>/todo add Build a starter house</code></td>
    </tr>
    <tr>
      <td><code>/todo list [open|all|completed] [page] [asc|desc]</code></td>
      <td>Lists available PlayerTodo entries.</td>
      <td><code>/todo list</code></td>
    </tr>
    <tr>
      <td><code>/todo page &lt;page&gt; [asc|desc]</code></td>
      <td>Runs the PlayerTodo page action.</td>
      <td><code>/todo page 1</code></td>
    </tr>
    <tr>
      <td><code>/todo view &lt;id&gt;</code></td>
      <td>Manages one of your PlayerTodo entries.</td>
      <td><code>/todo view 1</code></td>
    </tr>
    <tr>
      <td><code>/todo edit &lt;id&gt; &lt;new text&gt;</code></td>

    </tr>

    <tr>
      <td><code>/todo undo &lt;id&gt;</code></td>

      <td><code>/todo reopen &lt;id&gt;</code></td>

## Examples

- `/todo build a starter house near /warp town`
- `/todo add test copper blocks in the roof`
- `/todo list`
- `/todo list 5`
- `/todo list all 2`
- `/todo list 2 asc`
- `/todo list oldest`
- `/todo list open 2 desc`
- `/todo view 4`
- `/todo edit 4 test copper blocks in the greenhouse roof`
- `/todo search copper`
- `/todo done 4`
- `/todo undo 4`
- `/todo remove 4`
- `/todo remove 4 confirm`
- `/todo clear completed`
- `/todo share NikkiPixel 4`
- `/todo status`
