## How Players Use It

Start in-game with /todo. The sections below explain what PlayerTodo is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Open or view PlayerTodo with /todo.
- Check your own PlayerTodo progress or settings with /todo status.
- Browse available PlayerTodo entries with /todo list.

## Quick Start

1. Start with `/todo` to open or view PlayerTodo.
2. Use `/todo status` to check your current progress, settings, or state.
3. Use `/todo info` when you want the in-game help or a short explanation.
4. Try `/todo Build a starter house` when you are ready to use one of the feature actions.

- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
