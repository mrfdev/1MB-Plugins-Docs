This page introduces CMIProbe from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

CMIProbe is a lightweight test-server probe for server templates. It checks whether server and server library are present, whether selected server managers are reachable, whether server custom event classes can be discovered, and whether expected commands are registered after a build.

## How Players Use It

Start in-game with /cmiprobe. The sections below explain what CMIProbe is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Open or view CMIProbe with /cmiprobe.
- Check your own CMIProbe progress or settings with /cmiprobe status.

## Quick Start

1. Start with `/cmiprobe` to open or view CMIProbe.
2. Use `/cmiprobe status` to check your current progress, settings, or state.
3. Try `/cmiprobe managers` when you are ready to use one of the feature actions.

## Examples

- `/cmiprobe managers`
- `/cmiprobe firecheck`
- `/cmiprobe report`
- `/cmiprobe player mrfloris`
- `/cmiprobe joinwatch on all 180`
- `/cmiprobe joinwatch on SomePlayer 300`
- `/cmiprobe joinwatch status`
- `/cmiprobe joinwatch off`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
