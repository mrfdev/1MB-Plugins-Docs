This page introduces SparkReviewer from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

SparkReviewer is a server-management feature plugin for performance triage.

## How Players Use It

Start in-game with /sparkreviewer status. The sections below explain what SparkReviewer is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Captures live TPS, average MSPT when server exposes it, memory usage, player count, world count, and server library/Spark/MobFarmManager status.
- Counts loaded entities per world, including living entities, monsters, animals, item drops, projectiles, vehicles, hopper minecarts, tile entities, and hopper blocks in scanned loaded chunks.
- Lists chunk hotspots by entity/item/hopper/tile count.
- Dispatches Spark health and profiler commands from a single command surface.
- Writes named snapshots for before/after comparisons such as mfm-on versus mfm-off.
- Writes support-friendly dumps to the shared cache folder.
- Ships an offline Python companion script for copied server folders.

## Quick Start

1. Start with `/sparkreviewer status` to open or view SparkReviewer.
2. Use `/sparkreviewer status` to check your current progress, settings, or state.
3. Try `/sparkreviewer review` when you are ready to use one of the feature actions.

## Aliases

- `/sreview`
- `/spreview`

## Examples

- `/sparkreviewer review`
- `/sparkreviewer findings`
- `/sparkreviewer entities`
- `/sparkreviewer chunks`
- `/sparkreviewer logs`
- `/sparkreviewer mfm`
- `/sparkreviewer spark health`
- `/sparkreviewer spark start 180 farm-test`
- `/sparkreviewer stress mfm-on 180`
- `/sparkreviewer stress mfm-off 180`
- `/sparkreviewer compare mfm-on mfm-off`
- `/sparkreviewer dump`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- You may also see /sreview or /spreview as a shortcut.
- Top lists and leaderboards are public-facing progress views when the feature enables them.
