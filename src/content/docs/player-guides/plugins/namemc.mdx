This page introduces NameMC from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

NameMC lets players support the server on NameMC and verify that support in-game. Open the NameMC page, click the heart, then return to the server and run the verify command when you are ready.

## How Players Use It

Start with /namemc. The sections below explain what NameMC is for, which commands players can try, and what to expect when your account has access.

Available features include:

- A direct link from /namemc to the 1MoreBlock NameMC page.
- In-game verification of a NameMC server like for the matching Minecraft account.
- A configured one-time reward after successful verification.
- Persistent status showing whether the account has already verified its like.

## Quick Start

1. Start with `/namemc` to open or view NameMC.
2. Use `/namemc status` to check your current progress, settings, or state.
3. Use `/namemc info` when you want the in-game help or a short explanation.
4. Try `/namemc help` when you are ready to use one of the feature actions.

- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
