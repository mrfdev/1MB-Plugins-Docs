This page introduces ConsoleNoiseRouter from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

Server console message classification, redaction, and support dumps.

## How Players Use It

Start in-game with /consolenoise. The sections below explain what ConsoleNoiseRouter is for, which commands players can try, and what to expect when your account has access.

## Things You Can Do

- Open or view ConsoleNoiseRouter with /consolenoise.
- Check your own ConsoleNoiseRouter progress or settings with /consolenoise status.

## Quick Start

1. Start with `/consolenoise` to open or view ConsoleNoiseRouter.
2. Use `/consolenoise status` to check your current progress, settings, or state.
3. Try `/consolenoise patterns` when you are ready to use one of the feature actions.

## Examples

- `/consolenoise patterns`
- `/consolenoise recent warn`
- `/consolenoise recent all 2`
- `/consolenoise stats warn`
- `/consolenoise find deprecated`
- `/consolenoise dump error`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
