This page introduces MessageFont from a public server-docs point of view. It focuses on what the feature does in-game, how to get started when you have access, and which commands are useful.

## What It Does

MessageFont lets players choose a temporary style for private messages. It is meant as a cosmetic chat perk while still giving recipients a plain-mode option when they prefer easier-to-read messages.

## How Players Use It

Start with /msgfont. The sections below explain what MessageFont is for, which commands players can try, and what to expect when your account has access.

Available features include:

- A menu and paginated list for browsing available private-message font styles.
- Temporary personal font selection without raw formatting codes.
- Direct style selection for players who already know the style they want.
- A recipient-side plain mode that keeps incoming private messages easy to read.

## Quick Start

1. Start with `/msgfont` to open or view MessageFont.
2. Use `/msgfont status` to check your current progress, settings, or state.
3. Use `/msgfont help` when you want the in-game help or a short explanation.
4. Try `/msgfont list` when you are ready to use one of the feature actions.

## Aliases

- `/pmfont`

## Examples

- `/msgfont`
- `/msgfont list`
- `/msgfont set alt 30m`
- `/msgfont set uniform 2h`
- `/msgfont set illageralt 10m`
- `/msgfont clear`
- `/msgfont plain on`
- `/msgfont plain off`
- `/msgfont preview alt`

## Good To Know

- Some commands may be hidden until your group, event progress, or unlocks allow them.
- If a command opens a menu, follow the buttons in-game; the website is only the learning guide.
- You may also see /pmfont as a shortcut.
- Settings commands only change your own player preferences unless the page says otherwise.
