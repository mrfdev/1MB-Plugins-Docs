BSkyBlock is one of 1MoreBlock's BentoBox island game modes. It gives the mode its own world, island lifecycle, team controls, and progression hooks while sharing the common BentoBox add-ons documented elsewhere in this section.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Classic void-island survival and expansion.
- Broad compatibility with the BentoBox add-ons used across 1MoreBlock.
- Private island ownership with invitations, team membership, homes, and protection.
- Compatibility with shared Challenges, Level, Warps, Limits, Border, and other BentoBox add-ons.

## Commands

| Command | What it does |
| --- | --- |
| `/is` | Opens BSkyBlock help or the main island command. |
| `/is create` | Creates an island when creation is available. |
| `/is home` | Returns to your island home. |
| `/is team` | Shows the team-management commands. |
| `/is settings` | Opens island settings when enabled. |

## Getting started

1. Run `/is` and create an island, or open the 1MoreBlock game-types menu.
2. Read the starter messages and protect your starting resources.
3. Use the Challenges and Level pages to plan progression.

## Player notes

- Common aliases include `/is` and `/island`, but 1MoreBlock may customize aliases.
- Reset, team, coop, and visitor actions can be limited by cooldowns and rank permissions.
- Never accept an island invitation until you understand whether it moves or resets your current island progress.
