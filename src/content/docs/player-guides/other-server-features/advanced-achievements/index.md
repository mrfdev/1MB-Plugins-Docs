AdvancedAchievements turns normal server activity into a collection of long-term goals. Mining, crafting, travel, combat, economy, and other configured actions can unlock achievements and rewards.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Configurable achievements across many gameplay categories.
- Progress notifications, completion announcements, and reward delivery.
- Personal statistics, achievement lists, ranking views, and achievement books where enabled.
- 1MoreBlock-maintained compatibility fork for the current server runtime.

## Commands

| Command | What it does |
| --- | --- |
| `/aach` | Shows the AdvancedAchievements help. |
| `/aach list` | Opens or lists available achievements. |
| `/aach stats` | Shows your achievement statistics. |
| `/aach top` | Shows the achievement ranking. |
| `/aach book` | Requests the achievement book when that view is enabled. |
| `/aach week` | Shows weekly achievement activity when enabled. |
| `/aach month` | Shows monthly achievement activity when enabled. |

## Getting started

1. Run `/aach list` to see available goals.
2. Play normally and watch progress notifications.
3. Use `/aach stats` and `/aach top` to compare progress.

## Player notes

- Achievement names, thresholds, rewards, and hidden goals are configured by 1MoreBlock.
- Some categories can be disabled in specific worlds or game modes.
- Do not repeatedly automate an action solely to farm a reward; normal server and automation rules still apply.
