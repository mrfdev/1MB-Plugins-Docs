---
title: "MCPerks Guide"
description: "Offer configurable, time-limited or reusable gameplay perks through a menu."
---

MCPerks provides a configurable perk menu. A perk can grant a temporary effect or action, apply cooldowns, run reward logic, and require a particular rank or permission.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- GUI-driven perk selection.
- Configurable effects such as temporary flight, potion effects, or server-defined rewards.
- Cooldowns, queues, durations, and permission-gated perks.
- Optional mcMMO-aware perks and reward-command integration.

## Commands

| Command | What it does |
| --- | --- |
| `/mcperks` | Opens the available perks menu. |

## Getting started

1. Run `/mcperks`.
2. Read each perk description, duration, and cooldown.
3. Select only a perk available to your rank and current world.

## Player notes

- The exact perks on 1MoreBlock are configured and may differ from upstream examples.
- Perks can be unavailable in combat, restricted worlds, or conflicting states.
- A perk ending normally is not the same as losing permanent rank access.
