PyroMining adds RPG-like progression to naturally generated ores. Mining can award experience, Zeta, rare artifacts or gemstones, and encounters connected to vessels, requiems, rune guardians, thralls, and seasonal events.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Mining levels, XP, Zeta currency, statistics, and menus.
- Artifacts, gemstones, fluxes or current-version resources, and custom item drops.
- Vessels, requiems, thralls, bosses, rune guardians, and Oracle-related progression.
- Configurable worlds, ores, natural-block checks, cooldowns, and events.

## Commands

| Command | What it does |
| --- | --- |
| `/mine` | Shows PyroMining help or the main menu. |
| `/mine menu` | Opens the mining menu when exposed by the installed build. |
| `/mine stats` | Shows mining statistics. |
| `/mine artifacts` | Shows artifact information where enabled. |
| `/mine vessels` | Shows vessel information where enabled. |
| `/mine thrallcd` | Shows the current thrall/boss cooldown. |
| `/mine tcd` | Short alias for the thrall cooldown command. |

## Getting started

1. Run `/mine` and read the feature menus.
2. Mine naturally generated supported ores in an enabled world.
3. Inspect rare drops and read the official feature help before consuming or applying them.

## Player notes

- Placed ores generally do not count as naturally generated mining targets.
- Rare custom items and encounters can have cooldowns, level requirements, order requirements, or irreversible use.
- Zeta, XP, drop rates, enabled worlds, events, and current-version features are configured by 1MoreBlock.
