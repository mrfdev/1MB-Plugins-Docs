title: "AnimatedScoreboard Guide"
description: "Show a changing sidebar with useful server, player, world, and event information."
---

AnimatedScoreboard supplies the sidebar shown beside normal gameplay. Its lines can rotate, animate, and resolve placeholders so 1MoreBlock can surface useful information without filling chat.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Animated title and line transitions.
- PlaceholderAPI-backed player and server values.
- Different boards by world, permission, or context.
- Flicker-resistant scoreboard updates.

## Commands

This feature does not require a general player command. It appears automatically, through another menu, or through staff-created content.

## Getting started

1. Join a world where the scoreboard is enabled.
2. Watch for rotating values and status lines.
3. Report the exact world and a screenshot if the board is missing or stale.

## Player notes

- The board is display-only; most players do not need a command.
- Scoreboard content can differ by world, game mode, or rank.
- A client mod that replaces the scoreboard may hide the server board.
