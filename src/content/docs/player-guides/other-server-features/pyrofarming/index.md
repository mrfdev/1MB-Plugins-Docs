PyroFarming expands farming into its own progression system. Players can discover custom harvestables, use grow stations, build a codex, earn farming experience and Elysium, and interact with configured shops or markets.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Custom plants, bushes, harvestables, and grow stations.
- Farming levels, experience, Elysium currency, and progression.
- Codex discovery, farm menu, shop, market, and placed-item tracking.
- Farm friends and farming tournaments where enabled.

## Commands

| Command | What it does |
| --- | --- |
| `/farm` | Shows PyroFarming command help. |
| `/farm menu` | Opens the farming menu. |
| `/farm shop` | Opens the configured farming shop. |
| `/farm market` | Opens the farming market. |
| `/farm codex` | Opens your farming codex. |
| `/farm elysium` | Shows your Elysium balance. |
| `/farm placeditems` | Shows your tracked placed farming items. |
| `/farm friends` | Opens farm-friend settings when enabled. |

## Getting started

1. Run `/farm menu` and read the farming help/codex.
2. Obtain a configured starter crop or grow station.
3. Harvest normally and use the codex, shop, and progression menus to plan upgrades.

## Player notes

- Custom farming items contain plugin metadata; avoid renaming, stacking, or transforming them through unsupported mechanics.
- Market, shop, crop, XP, and Elysium values are configured by 1MoreBlock.
- Farm friends is optional and does not replace land/island/plot protection.
