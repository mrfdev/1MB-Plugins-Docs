---
title: "SelectionVisualizer Guide"
description: "Show authorized builders the boundaries, points, and size of their WorldEdit selections."
---

SelectionVisualizer makes a WorldEdit selection visible with particles or outlines. It helps authorized builders understand exactly which area a command will affect before they run a large edit.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Visual first and second selection points.
- Particle or outline rendering around cuboid and supported selection shapes.
- Selection size and update feedback.
- WorldEdit/FAWE integration without changing the selection itself.

## Commands

This feature does not require a general player command. It appears automatically, through another menu, or through staff-created content.

## Getting started

1. Receive authorized WorldEdit access.
2. Set both selection positions with the normal WorldEdit tools.
3. Inspect the visible boundary before running an edit.

## Player notes

- This feature is for authorized builders, not normal survival selection or claims.
- Client particle settings can make the outline hard to see.
- The visual boundary does not itself protect or modify blocks.
