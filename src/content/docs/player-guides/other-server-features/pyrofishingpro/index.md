PyroFishingPro turns fishing into a deeper progression system. Custom fish have tiers and values, while Entropy, augments, skills, deliveries, totems, bait, shops, and tournaments provide long-term goals.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Custom fish species, tiers, weights, biomes, and rarity.
- Fishing levels, XP, Entropy, skills, augments, and rod progression.
- Fish bag, codex, shop, scales, gutting station, deliveries, bait, and boosters.
- Scheduled tournaments and fishing totems where enabled.

## Commands

| Command | What it does |
| --- | --- |
| `/fish` | Shows fishing command help. |
| `/fish menu` | Opens the main fishing menu. |
| `/fish codex` | Shows discovered and available fish. |
| `/fish bag` | Opens the fish bag. |
| `/fish shop` | Opens the fish shop. |
| `/fish scales` | Opens fish scaling. |
| `/fish gut` | Opens the gutting station. |
| `/fish deliveries` | Opens delivery progress and rewards. |
| `/fish augment` | Opens the augment application menu. |
| `/fish augments` | Lists available augments. |
| `/fish entropy` | Shows your Entropy balance. |
| `/fish skills` | Opens fishing skills. |
| `/ft time` | Shows time until the next tournament when enabled. |

## Getting started

1. Run `/fish menu` and `/fish codex`.
2. Fish in allowed water and inspect custom catches before selling or gutting.
3. Save Entropy and skill points for deliberate upgrades.

## Player notes

- Custom fish are metadata-bearing items; ordinary fish and custom fish behave differently.
- Shop prices, biomes, tournament schedule, delivery thresholds, and enabled systems are server configured.
- Gutting, scaling, selling, and augmenting can consume or permanently alter items.
