VotingPlugin handles votes received from configured server-list sites. It can show vote links, deliver rewards, track totals and streaks, run milestones, power a vote shop, and contribute to community events.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Vote-site menu and clickable vote links.
- Immediate, offline, cumulative, streak, milestone, and top-voter rewards.
- Vote points, vote shop, totals, history, reminders, and rankings.
- Votifier/NuVotifier listener integration and placeholders.

## Commands

| Command | What it does |
| --- | --- |
| `/vote` | Opens vote links and vote information. |
| `/vote total` | Shows your vote total. |
| `/vote points` | Shows vote points where enabled. |
| `/vote shop` | Opens the vote shop where enabled. |
| `/vote top` | Shows top voters. |
| `/vote last` | Shows recent voting information where enabled. |

## Getting started

1. Run `/vote`.
2. Open an official listed site and follow its voting form.
3. Return to the server and allow time for the site to deliver the vote.

## Player notes

- Only use vote links published by 1MoreBlock.
- Each voting site controls its own account, captcha, name, and cooldown rules.
- A vote-site confirmation does not prove the server received the callback; keep the site and time for support.
