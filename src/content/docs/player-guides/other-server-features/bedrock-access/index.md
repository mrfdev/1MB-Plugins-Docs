title: "Bedrock Access (Geyser and floodgate) Guide"
description: "Allow supported Minecraft Bedrock clients to connect to the Java server through Geyser and floodgate."
---

Geyser translates Bedrock Edition network traffic so supported Bedrock clients can join the Java Edition 1MoreBlock server. floodgate supplies secure Bedrock identity handling without requiring every Bedrock player to own or sign into a Java account.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Bedrock-to-Java protocol translation.
- floodgate identity and authentication integration.
- Cross-platform movement, inventory, chat, forms, skins, and gameplay translation.
- Optional account linking according to 1MoreBlock policy.

## Commands

This feature does not require a general player command. It appears automatically, through another menu, or through staff-created content.

## Getting started

1. Add the Bedrock address and port published by 1MoreBlock.
2. Join with a supported, up-to-date Bedrock client.
3. Use the name shown in game when asking staff for account help.

## Player notes

- Java and Bedrock are different clients; some UI, combat, redstone, or interaction behavior can differ.
- The published Bedrock port may differ from the Java port.
- Never share account-linking codes or authentication details.
