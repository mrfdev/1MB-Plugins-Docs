title: "TryMe Guide"
description: "Run short chat and gameplay challenges that players can answer or complete for configured rewards."
---

TryMe is a configurable mini-challenge system. It can announce quick tasks, trivia, typing challenges, math, reactions, or other game types and reward the first or qualifying players who complete them.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Multiple chat and action challenge types.
- Random or scheduled games with hints and time limits.
- Winner announcements, streaks, statistics, and configured rewards.
- Custom questions, words, math ranges, and game rotation.

## Commands

| Command | What it does |
| --- | --- |
| `/tryme` | Shows TryMe help or player information. |
| `/tryme stats` | Shows challenge statistics where enabled. |
| `/tryme top` | Shows challenge rankings where enabled. |

## Getting started

1. Watch chat for a TryMe announcement.
2. Read the exact challenge and answer in the requested format.
3. Wait for the winner/result message before repeating an answer.

## Player notes

- Answer channels, capitalization, time limit, and winner count depend on the active game.
- Do not use automated chat, macros, or scripts to answer challenges.
- Rewards and schedule are configured by 1MoreBlock.
