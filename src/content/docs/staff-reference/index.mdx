## Custom Server Plugins

- [1MB-XRayHunter](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/1mb-xrayhunter/README.md) - Staff-facing CoreProtect add-on for suspicious mining lookups, cached ore detail review, and large-archive X-ray triage.
- [AntiPopup](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/antipopup/README.md) - Hide the unsafe-server popup and limit vanilla chat reporting on 1MoreBlock.
- [1MB-AntiXRay](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/coreprotect-antixray/README.md) - Review CoreProtect ore-break patterns, compare players, and investigate suspicious mining activity.
- [Summer Floaties](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/eventfloaties/README.md) - Build and verify summer floatie decorations for event rewards.
- [HeadDisplays](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/headdisplays/README.md) - Staff tool for floating head-display labels, documented here as a patched third-party compatibility artifact for Paper 26.2 testing.
- [Lootbox](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/lootbox/README.md) - Discover repeatable loot containers with randomized rewards on 1MoreBlock.
- [1MB-MapHide](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/maphide/README.md) - Lets players control whether their live BlueMap marker is visible.
- [mcMMO Discord Alerts](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/mcmmo-discord-alerts/README.md) - Announces 1MoreBlock mcMMO power-level milestones in Discord with compact skill summaries.; [official documentation](https://docs.discordsrv.com/alerts/)
- [1MB mcMMO More](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/mcmmo-more/README.md) - Adds 1MoreBlock-owned grindable skills alongside mcMMO without patching mcMMO itself.
- [PlotMarkers](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/plotmarkers/README.md) - Shows PlotSquared plot owners and boundaries on BlueMap.
- [ShopChest](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/shopchest/README.md) - Create persistent player chest shops with holograms and Vault economy trading.
- [XMas Tree](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/xmas-tree/README.md) - Plant, grow, and celebrate upgradeable Christmas trees with presents, streaks, milestones, and preserved legacy event trees.
- [1MB XRayHeuristics](https://github.com/mrfdev/1MB-Plugins-Docs/blob/main/project-docs/xrayheuristics/README.md) - Heuristic Anti-XRay add-on that tracks suspicious mining patterns and staff handling workflows.

## Update flow

`npm run docs:sync` refreshes only the CMI namespace. Use the registered project importer for standalone projects; curated third-party feature notes live under `catalog/other-server-features/`. Every update regenerates and validates the complete site before deployment.
