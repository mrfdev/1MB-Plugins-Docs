# BentoBox FarmersDance

BentoBox FarmersDance is a BentoBox add-on used by one or more 1MoreBlock island game modes. It extends an island rather than creating a separate server world of its own.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Movement-driven crop growth for supported plants.
- Configurable chance, range, particles, and game-mode hooks.
- A playful alternative to waiting for normal random ticks.

## Commands

This feature does not require a general player command. It appears automatically, through another menu, or through staff-created content.

## Getting started

1. Enter the relevant island game mode first.
2. The feature appears automatically when its conditions are met.
3. Check the game-mode help if the command label differs, because BentoBox commands can be customized per mode.

## Player notes

- Availability, prices, cooldowns, limits, and rewards are controlled by 1MoreBlock configuration.
- An add-on may be enabled for one island mode but not another.
- Use the command root for your current island mode, such as `/is`, `/ob`, `/acid`, `/cave`, or `/skygrid`.
