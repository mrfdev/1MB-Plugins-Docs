# BentoBox MagicCobblestoneGenerator

BentoBox MagicCobblestoneGenerator is a BentoBox add-on used by one or more 1MoreBlock island game modes. It extends an island rather than creating a separate server world of its own.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Tiered or permission-based generator recipes.
- Weighted block distributions by game mode and world.
- Island progression rewards that can unlock better generator output.

## Commands

| Command | What it does |
| --- | --- |
| `/is generator` | Shows generator information in modes exposing this command. |
| `/ob generator` | Shows OneBlock generator information when enabled. |

## Getting started

1. Enter the relevant island game mode first.
2. Use `/is generator` or `/ob generator` to open the feature.
3. Check the game-mode help if the command label differs, because BentoBox commands can be customized per mode.

## Player notes

- Availability, prices, cooldowns, limits, and rewards are controlled by 1MoreBlock configuration.
- An add-on may be enabled for one island mode but not another.
- Use the command root for your current island mode, such as `/is`, `/ob`, `/acid`, `/cave`, or `/skygrid`.
