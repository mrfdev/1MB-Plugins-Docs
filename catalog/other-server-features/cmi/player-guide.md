# CMI

CMI is a broad server-management plugin that supplies many everyday 1MoreBlock commands and APIs. Player access is intentionally permission-based, so the commands available to one rank or world can differ from another.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Homes, warps, spawn, teleport requests, back, and location tools.
- Economy, kits, mail, nicknames, chat, and utility commands.
- Player information, cooldowns, command costs, portals, holograms, and scheduled actions.
- Events and APIs used heavily by the custom 1MB-CMIAPI feature plugins.

## Commands

| Command | What it does |
| --- | --- |
| `/cmi` | Shows CMI help filtered to commands you can use. |
| `/cmi help <command>` | Shows help for a CMI command. |
| `/home` | Opens or uses your available homes. |
| `/warp` | Shows or uses available server warps. |
| `/tpa <player>` | Requests teleportation when enabled. |
| `/back` | Returns to an available previous location. |
| `/balance` | Shows your economy balance. |

## Getting started

1. Run `/cmi` or the shorter command you already know.
2. Use `/cmi help <command>` for syntax.
3. Read cooldown, cost, and confirmation messages before repeating a command.

## Player notes

- Short aliases can be owned by CMI or by a 1MoreBlock feature; trust the live help output.
- Some commands have costs, cooldowns, world restrictions, or target permissions.
- A command shown in upstream CMI documentation is not automatically granted on 1MoreBlock.
