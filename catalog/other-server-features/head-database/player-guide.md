# HeadDatabase

HeadDatabase provides a searchable menu of decorative Minecraft heads. On 1MoreBlock it can be used for detailed builds, labels, furniture, seasonal decoration, and staff-created content where access is granted.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Categorized GUI containing decorative heads.
- Text search across head names and categories.
- Player-head lookup and favorites in supported builds.
- Economy or permission controls for taking heads.

## Commands

| Command | What it does |
| --- | --- |
| `/hdb` | Opens the head database when you have access. |
| `/hdb search <term>` | Searches for heads by name. |
| `/hdb info` | Shows HeadDatabase information in supported builds. |

## Getting started

1. Run `/hdb`.
2. Browse a category or search for a short descriptive term.
3. Select a head and read any cost or permission message.

## Player notes

- Availability and price can vary by world or rank.
- Decorative heads are still items and follow normal item-loss and placement rules.
- Search terms and menu layout depend on the currently downloaded database.
