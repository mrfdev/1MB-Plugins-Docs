# SkyGrid

SkyGrid is one of 1MoreBlock's BentoBox island game modes. It gives the mode its own world, island lifecycle, team controls, and progression hooks while sharing the common BentoBox add-ons documented elsewhere in this section.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Scattered blocks create traversal and resource-recovery challenges.
- Island ownership and protection apply to each SkyGrid play area.
- Private island ownership with invitations, team membership, homes, and protection.
- Compatibility with shared Challenges, Level, Warps, Limits, Border, and other BentoBox add-ons.

## Commands

| Command | What it does |
| --- | --- |
| `/skygrid` | Opens SkyGrid help or the main island command. |
| `/skygrid create` | Creates an island when creation is available. |
| `/skygrid home` | Returns to your island home. |
| `/skygrid team` | Shows the team-management commands. |
| `/skygrid settings` | Opens island settings when enabled. |

## Getting started

1. Run `/skygrid` and create an island, or open the 1MoreBlock game-types menu.
2. Read the starter messages and protect your starting resources.
3. Use the Challenges and Level pages to plan progression.

## Player notes

- Common aliases include `/skygrid` and `/sg`, but 1MoreBlock may customize aliases.
- Reset, team, coop, and visitor actions can be limited by cooldowns and rank permissions.
- Never accept an island invitation until you understand whether it moves or resets your current island progress.
