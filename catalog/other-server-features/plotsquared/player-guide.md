# PlotSquared

PlotSquared manages designated plot worlds. Players can claim or auto-assign a plot, build inside it, invite collaborators, adjust allowed settings, and travel between plot homes.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Automatic and manual plot claiming.
- Owners, members, trusted users, denied users, aliases, and homes.
- Plot flags, merging, biome changes, clearing, and schematic tools where permitted.
- FAWE-backed plot generation and large edit operations.

## Commands

| Command | What it does |
| --- | --- |
| `/plot auto` | Claims and teleports to an available plot. |
| `/plot claim` | Claims the plot you are standing in when available. |
| `/plot home` | Teleports to your plot home. |
| `/plot info` | Shows the current plot owner and settings. |
| `/plot add <player>` | Allows a player to build while the owner is online, depending on settings. |
| `/plot trust <player>` | Grants broader trusted access. |
| `/plot remove <player>` | Removes member/trusted access. |
| `/plot visit <player>` | Visits a player's public plot. |

## Getting started

1. Enter the plot world and run `/plot auto`.
2. Use `/plot info` to confirm ownership.
3. Add collaborators carefully and set a useful plot alias/home.

## Player notes

- `add` and `trust` can grant different levels of access.
- Clear, delete, unlink, and merge actions can permanently change a plot.
- Plot rules and flags do not override server-wide behavior unless explicitly supported.
