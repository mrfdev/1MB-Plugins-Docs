# LuckPerms

LuckPerms is the permissions platform that decides which commands and features each player, rank, world, or context may use. Players normally encounter its results through ranks and command access rather than through LuckPerms commands.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- User and group permissions with inheritance.
- World, server, time, and other contextual permission rules.
- Prefixes, suffixes, tracks, temporary grants, and promotion paths.
- Web editor and detailed permission-check diagnostics for authorized staff.

## Commands

This feature does not require a general player command. It appears automatically, through another menu, or through staff-created content.

## Getting started

1. Use the commands and menus associated with your current rank.
2. Read a denial message carefully; it may identify a missing permission or contextual restriction.
3. Contact staff with the exact command, world, and time if access seems wrong.

## Player notes

- A visible rank name does not guarantee every upstream plugin permission.
- Permissions can differ by world, game mode, or temporary context.
- LuckPerms is not a player rank-up command unless 1MoreBlock exposes a separate promotion workflow.
