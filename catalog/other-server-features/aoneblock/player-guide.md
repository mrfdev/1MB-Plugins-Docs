# AOneBlock

AOneBlock is one of 1MoreBlock's BentoBox island game modes. It gives the mode its own world, island lifecycle, team controls, and progression hooks while sharing the common BentoBox add-ons documented elsewhere in this section.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- A regenerating central block advances through configured phases.
- Blocks, mobs, containers, and rare events expand as phase progress increases.
- Private island ownership with invitations, team membership, homes, and protection.
- Compatibility with shared Challenges, Level, Warps, Limits, Border, and other BentoBox add-ons.

## Commands

| Command | What it does |
| --- | --- |
| `/ob` | Opens AOneBlock help or the main island command. |
| `/ob create` | Creates an island when creation is available. |
| `/ob home` | Returns to your island home. |
| `/ob team` | Shows the team-management commands. |
| `/ob settings` | Opens island settings when enabled. |

## Getting started

1. Run `/ob` and create an island, or open the 1MoreBlock game-types menu.
2. Read the starter messages and protect your starting resources.
3. Use the Challenges and Level pages to plan progression.

## Player notes

- Common aliases include `/ob` and `/oneblock`, but 1MoreBlock may customize aliases.
- Reset, team, coop, and visitor actions can be limited by cooldowns and rank permissions.
- Never accept an island invitation until you understand whether it moves or resets your current island progress.
