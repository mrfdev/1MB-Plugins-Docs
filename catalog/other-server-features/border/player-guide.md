# BentoBox Border

BentoBox Border is a BentoBox add-on used by one or more 1MoreBlock island game modes. It extends an island rather than creating a separate server world of its own.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Per-island border visualization.
- Protection against crossing outside the owned island area where configured.
- Border size follows island range and game-mode rules.

## Commands

| Command | What it does |
| --- | --- |
| `/is border` | Toggles or shows your island border when enabled. |
| `/ob border` | Toggles the OneBlock island border when enabled. |

## Getting started

1. Enter the relevant island game mode first.
2. Use `/is border` or `/ob border` to open the feature.
3. Check the game-mode help if the command label differs, because BentoBox commands can be customized per mode.

## Player notes

- Availability, prices, cooldowns, limits, and rewards are controlled by 1MoreBlock configuration.
- An add-on may be enabled for one island mode but not another.
- Use the command root for your current island mode, such as `/is`, `/ob`, `/acid`, `/cave`, or `/skygrid`.
