# ajLeaderboards

ajLeaderboards turns numeric PlaceholderAPI values into ordered rankings. On 1MoreBlock those rankings can appear in menus, signs, holograms, or other displays without every source plugin needing its own leaderboard system.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Placeholder-driven rankings for supported statistics.
- Offline and cached leaderboard values where configured.
- Rank, player-name, score, size, reset-time, and position placeholders.
- Display integrations for signs and other server UI.

## Commands

This feature does not require a general player command. It appears automatically, through another menu, or through staff-created content.

## Getting started

1. Look for leaderboard displays in the relevant game or server area.
2. Use the source plugin, such as Jobs or mcMMO, to inspect your underlying score.
3. Allow cache updates time to reach public displays.

## Player notes

- There is no general player command required to appear on a leaderboard.
- Rankings may update on a schedule rather than immediately.
- Different boards can use different time periods, filters, and score formats.
