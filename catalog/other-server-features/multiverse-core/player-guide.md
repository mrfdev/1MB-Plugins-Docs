# Multiverse-Core

Multiverse-Core is the multi-world platform used to load, create, import, unload, teleport between, and configure worlds. Players usually access worlds through 1MoreBlock menus and warps rather than Multiverse administration commands.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Multiple loaded worlds with separate properties.
- World aliases, spawn locations, environments, difficulties, and game rules.
- World teleport destinations and portal/plugin integration.
- Foundation for Multiverse-Inventories world groups.

## Commands

This feature does not require a general player command. It appears automatically, through another menu, or through staff-created content.

## Getting started

1. Use the 1MoreBlock menu, game-type selector, or public warp.
2. Check the destination name and rules before teleporting.
3. Report world-transfer issues with source and destination world names.

## Player notes

- A world can have different inventory, game mode, difficulty, and protection.
- BentoBox manages its own game-mode worlds on top of the server world system.
- Direct Multiverse teleport commands are staff-only unless explicitly exposed.
