# PyroWelcomesPro

PyroWelcomesPro controls selected welcome and presence messages around players joining or leaving 1MoreBlock. It can combine formatting, placeholders, sounds, titles, and optional Discord integration into a consistent arrival experience.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Join, first-join, and leave presentation rules.
- MiniMessage or configured rich text, sounds, titles, and actions.
- PlaceholderAPI-backed player/server information.
- Optional DiscordSRV message integration.

## Commands

This feature does not require a general player command. It appears automatically, through another menu, or through staff-created content.

## Getting started

1. Join normally and read the welcome information shown.
2. Use the linked help/menu commands rather than repeatedly reconnecting to retrieve a tip.
3. Report an incorrect message with a screenshot and whether it was first join, join, or leave.

## Player notes

- This is primarily an automatic presentation feature.
- Messages can differ for first join, returning players, ranks, worlds, or vanished staff.
- Discord and Minecraft may intentionally use different wording.
