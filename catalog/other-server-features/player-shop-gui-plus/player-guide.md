# PlayerShopGUIPlus

PlayerShopGUIPlus is the player-to-player marketplace. Sellers create listings from their own items; buyers browse a shared GUI and pay the listing price to the seller, subject to 1MoreBlock taxes and limits.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Shared GUI for player listings.
- Sell, browse, search, inspect, buy, cancel, and expired-listing workflows.
- Configurable listing limits, durations, taxes, refunds, worlds, and game modes.
- Seller balances and offline transaction handling through the configured economy.

## Commands

| Command | What it does |
| --- | --- |
| `/playershop` | Opens the player marketplace. |
| `/playershop sell <price>` | Lists the held item at a price when permitted. |
| `/playershop player <name>` | Shows a player's listings. |
| `/playershop cancel` | Opens or performs your listing cancellation flow. |

## Getting started

1. Open `/playershop` and inspect existing listings.
2. To sell, hold the exact item and use `/playershop sell <price>`.
3. Review quantity, metadata, price, tax, and confirmation before listing or buying.

## Player notes

- PlayerShopGUIPlus is for player listings; ShopGUIPlus is the server shop.
- Renamed, enchanted, damaged, container, or custom items must be inspected carefully.
- Taxes, listing limits, expiry, prohibited items, and allowed worlds are server configured.
