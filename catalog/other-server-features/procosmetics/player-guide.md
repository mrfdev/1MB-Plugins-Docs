# ProCosmetics

ProCosmetics provides optional visual and playful effects without replacing core progression. Available categories can include gadgets, particles, pets, morphs, mounts, music, hats, death effects, and treasure chests.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- GUI-based cosmetic browsing and equipping.
- Hundreds of configurable cosmetics across multiple categories.
- Coins, ammo, treasure chests, rarity, unlocks, and permissions.
- Self-view controls and world/game-state restrictions.

## Commands

| Command | What it does |
| --- | --- |
| `/cosmetics` | Opens the main cosmetics menu. |
| `/equip <category> <cosmetic>` | Equips an unlocked cosmetic. |
| `/unequip` | Unequips a selected cosmetic type. |
| `/unequipall` | Removes all equipped cosmetics. |
| `/toggleselfview <type>` | Toggles your own view of supported cosmetics. |

## Getting started

1. Run `/cosmetics`.
2. Choose a category and inspect unlocked items.
3. Equip a cosmetic and use `/unequipall` if you want to clear everything.

## Player notes

- Cosmetics can be disabled in PvP, restricted worlds, minigames, or performance-sensitive areas.
- Unlocks, coins, ammo, and treasure rewards are configured by 1MoreBlock.
- Cosmetic effects do not grant permission to disrupt other players or protected areas.
