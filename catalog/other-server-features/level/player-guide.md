# BentoBox Level

BentoBox Level is a BentoBox add-on used by one or more 1MoreBlock island game modes. It extends an island rather than creating a separate server world of its own.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- Scans configured island blocks and applies value rules.
- Personal island-level checks and top-island rankings.
- Death penalties, visit requirements, and calculation cooldowns where configured.

## Commands

| Command | What it does |
| --- | --- |
| `/is level` | Calculates or shows your SkyBlock island level. |
| `/is top` | Shows top SkyBlock islands. |
| `/ob level` | Calculates or shows your OneBlock island level. |

## Getting started

1. Enter the relevant island game mode first.
2. Use `/is level` or `/is top` to open the feature.
3. Check the game-mode help if the command label differs, because BentoBox commands can be customized per mode.

## Player notes

- Availability, prices, cooldowns, limits, and rewards are controlled by 1MoreBlock configuration.
- An add-on may be enabled for one island mode but not another.
- Use the command root for your current island mode, such as `/is`, `/ob`, `/acid`, `/cave`, or `/skygrid`.
