# MiniMOTD

MiniMOTD controls what players see for 1MoreBlock before connecting: the two-line server-list message, colors and formatting, player-count display, hover samples, and favicon behavior.

> Commands and features on 1MoreBlock depend on the current server configuration, world, and your permissions. A command listed here may be limited to certain ranks or contexts.

## What it adds

- MiniMessage-formatted server-list description.
- Configurable player-count and maximum-player display.
- Hover/sample text and favicon selection.
- Multiple MOTDs or icons with weighted/random selection.

## Commands

This feature does not require a general player command. It appears automatically, through another menu, or through staff-created content.

## Getting started

1. Add 1MoreBlock to the multiplayer server list.
2. Refresh the list to see the current message and player count.
3. Hover the player count to inspect any configured sample text.

## Player notes

- The server-list result can be cached briefly by the client or proxy.
- The shown maximum can be a presentation value rather than a hard capacity.
- MiniMOTD affects the pre-login server list, not in-game tab or chat.
